Меню:
* выполнение функции 6
* выполнение функции 7
* выполнение функции 10
* выход из программы

В данной программе реализованы формулы:
- [x] $ln(1 + x)$
- [x] $ln(1 - x)$
- [ ] $(1 - x)^m$
